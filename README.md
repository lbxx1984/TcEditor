
# Using webpack

## Install
```bash
npm install
```

## Startup
```bash
npm start
```

## Visit:
[http://localhost:8088/](http://localhost:8088/)

## Build
npm run build

# Using edp

## Install
```bash
npm install edp -g
npm install
```

## Startup
```bash
edp webserver
```

## Visit
[http://localhost:8089/](http://localhost:8089/)
